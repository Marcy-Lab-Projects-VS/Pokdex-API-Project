# Pokédex

## Application Overview
The Pokédex application is a straightforward tool that allows users to search for Pokémon and view essential details. It uses the Pokémon API to fetch and display information about each Pokémon.

## Pokémon Search
Users can search for Pokémon by name.

## Pokémon User Input Fetched
After submitting a search query, the application fetches and displays the Pokémon's front and back sprites, its shiny variant, name, and ID number.
